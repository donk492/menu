# Cocktail Menu

A handcrafted cocktail menu website with elegant sketch-style design.

## Features

- 17 Short Drinks
- 7 Long Drinks
- 1 Shot
- 3 Straight Spirits
- Hidden secret drink (Fake Denki Bran)
- Beautiful animations and hover effects
- Responsive design
